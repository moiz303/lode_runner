Мой проект - игра из 80-х Lode Runner.
Задача в игре - собрать все монетки, разложенные по уровню.
Уровни, как и монетки, располагаются в несколько "слоёв", между которыми можно перемещаться только по лестницам.
Сам уровень может быть как выбран из уже имеющихся в базе, так и сгенерирован отдельно, но он может получиться невозможным.
Монетки тоже раскладываются по уровню случайным способом.
Ещё есть роботы, которые будут мешать сбору монеток - бежать прямо к вам по самому короткому пути.
Уровень заканчивается, когда будут собраны все монетки, потом будет выведено время, за которое был пройден уровень.
